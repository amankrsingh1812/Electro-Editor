Electro-Editor
======

A custom text-editor made with ElectronJS and HTML that uses in-memory Piece Table.

## Features Implemented
* Undo & Redo
* Copy & Paste
* Find & Replace
* File Backup
* In-app Terminal
* Automatic Reload
