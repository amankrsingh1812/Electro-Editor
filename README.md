Electro-Editor
======

A custom text-editor made with electron-js that uses in-memory Piece Table.

## Features Implemented
* Undo & Redo
* Copy & Paste
* Find & Replace
* File Backup
* In-app Terminal
* Automatic Reload

## To run

 ```

 cd ElectronJs-Project/
 npm start
 ```
